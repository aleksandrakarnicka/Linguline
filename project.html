Who are we?

We are natural language processing (NLP) students who took part in a web development course

What is the goal of the project?

The goal of the project was to utilise web development tools and to build a website that would consist of NLP-related implementations

What can the chatbot do?

The chatbot is able to both read and listen to the input, based on the user's preference. More importantly, it answers any queries about words that are in the dictionary

What tools we used?

We used: JavaScript, HTML, CSS, Free Dictionary API
